# College-LMS-Portal

A full-stack Learning Management System (LMS) web application designed for academic institutions to manage courses, instructors, students, assignments, submissions, and grades efficiently. This project demonstrates complete end-to-end web application development with a modern UI, secure backend, and scalable architecture suitable for real-world academic environments.

## ðŸ›  Setup Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/Adityasingh6996/College-LMS-Portal.git
   ```

2. Install frontend dependencies:
   ```bash
   cd Client
   npm install
   ```

3. Install backend dependencies:
   ```bash
   cd ../Server
   npm install
   ```

4. Configure `.env` file (DB URI, JWT secret, etc.)

5. Run the project:
   ```bash
   npm start      # Frontend
   npm run server # Backend
   ```
